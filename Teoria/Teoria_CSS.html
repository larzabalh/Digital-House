Los posicionamiento son para mover un poco las cosas

Positicion relative, mueve relativamente desde la posicion que deberia estar. Va siempre de top,left,right,botton.

Posicionamiento absoluto: lo mueve respecto al principio del body. Son para hacer barras de navegacion

Position FIX: Se queda fijo cuando SCROLLEO!!!

Position absolut, pero la caja padre tiene que estar en relative

Z INDEX: va siempre con el position relative: segun el numero mayor, aparece al frente o detras de otro elemento.

CENTRAR HORIZONTALMENTE: Para centrar y la caja tiene que tener width, margin = 0 auto y tiene que ser display: Block

/* width: 500px; margin: 0 auto; display: block; */

CENTRAR VERTICALMENTE: positicion relative, top:50% y left: 50% (aca se centra la caja) y ahora hay que usar TRANSFORM: TRANSLATE(-50%,50%) /* position: relative; top: 50%; left: 50%; transform: translate(-50%,50%); */

position: relative; top: 50%; left: 50%; transform: translate(-50%,50%);
